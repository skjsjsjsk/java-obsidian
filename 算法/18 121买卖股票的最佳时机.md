- 维护一个ans作为结果, 初始化为0
- 维护一个minPrice更新每轮循环时的最小价格, 直到循环结束找到了最小价格
- 
```
  class Solution {
    public int maxProfit(int[] prices) {
        int ans = 0;
        int minPrice = prices[0];
        for(int n: prices){
            ans = Math.max(ans, n - minPrice);
            minPrice = Math.min(n, minPrice);
        }
        return ans;
    }
}
```
