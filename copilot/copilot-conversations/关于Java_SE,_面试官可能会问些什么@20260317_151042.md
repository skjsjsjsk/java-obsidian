**ai**: 针对 Java SE（Java Standard Edition），面试官通常会从基础语法、集合框架、多线程并发、JVM 以及新特性这几个核心维度进行考察。考虑到你正在准备校招或社招，以下是我整理的高频面试考点清单，可以作为你填充 [[Java SE]] 笔记的大纲：

### 1. Java 基础
- **基本数据类型与包装类**
  - `int` 和 `Integer` 的区别？什么是自动拆装箱？
  - `Integer` 缓存机制（`Integer.valueOf`）的范围是多少？
- **String**
  - String 为什么是不可变的？
  - String、StringBuffer、StringBuilder 的区别？
  - `new String("abc")` 创建了几个对象？
  - 字符串常量池（String Pool）在哪里？JDK 1.7 之后发生了什么变化？
- **面向对象**
  - 重写（Override）和重载（Overload）的区别？
  - 接口和抽象类的区别？（JDK 1.8 之后接口可以有默认方法，这点很重要）
  - 深拷贝与浅拷贝的区别？
- **其他基础**
  - `==` 和 `equals()` 的区别？
  - Java 中的异常体系结构？Error 和 Exception 的区别？Checked Exception 和 Unchecked Exception？
  - Java 的反射机制及其优缺点？

### 2. 集合框架 - **重中之重**
- **ArrayList**
  - ArrayList 的扩容机制？（默认初始大小，扩容倍数）
  - ArrayList 和 LinkedList 的区别？（基于数组 vs 双向链表）
- **HashMap** - **超级高频**
  - HashMap 的底层数据结构？（JDK 1.8 数组+链表+红黑树）
  - HashMap 的 put 过程详解？
  - HashMap 的扩容机制？为什么是 2 的 n 次幂？
  - HashMap 是线程安全的吗？如何解决线程安全问题？
- **ConcurrentHashMap**
  - JDK 1.7（分段锁 Segment）与 JDK 1.8（CAS + synchronized）实现的区别？
- **HashSet**
  - HashSet 的底层实现？如何保证元素不重复？

### 3. 多线程与并发 (JUC) - **重中之重**
- **线程基础**
  - 进程和线程的区别？
  - 创建线程的几种方式？
  - 线程的生命周期（NEW, RUNNABLE, BLOCKED, WAITING, TIMED_WAITING, TERMINATED）？
- **锁机制**
  - `synchronized` 关键字的底层原理？（对象头 Mark Word、Monitor）
  - 锁升级过程？（偏向锁 -> 轻量级锁 -> 重量级锁）
  - `volatile` 关键字的作用？（可见性、禁止指令重排，不保证原子性）
  - `synchronized` 和 `ReentrantLock` 的区别？
- **线程池**
  - 线程池的 7 个核心参数？
  - 线程池的工作流程？
  - 拒绝策略有哪些？
  - 为什么不建议使用 `Executors` 创建线程池？
- **并发工具与原理**
  - `ThreadLocal` 的原理？Key 为什么是弱引用？内存泄漏问题如何解决？
  - CAS (Compare And Swap) 原理及 ABA 问题如何解决？
  - AQS (AbstractQueuedSynchronizer) 原理简述？

### 4. JVM (Java Virtual Machine)
- **内存模型 (JMM)**
  - JVM 运行时数据区包含哪些部分？（堆、栈、方法区、程序计数器、本地方法栈）
  - 堆内存划分？
- **垃圾回收 (GC)**
  - 如何判断对象可以被回收？（引用计数法 vs 可达性分析法）
  - GC 算法有哪些？（标记-清除、标记-整理、复制算法）
  - 常见的垃圾收集器？（CMS, G1, ZGC）
- **类加载机制**
  - 类加载的过程？（加载、验证、准备、解析、初始化）
  - 什么是双亲委派模型？为什么要这样设计？

### 5. Java 新特性
- **Java 8**
  - Lambda 表达式与函数式接口。
  - Stream API 的常用操作。
  - Optional 类解决空指针异常。
- **Java 其他版本**
  - JDK 17/21 的新特性（如 Records, Switch 模式匹配, 虚拟线程等，视岗位要求而定）。
