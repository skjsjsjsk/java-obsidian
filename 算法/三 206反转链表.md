-  在链表里面, 返回链表的头节点就是返回链表本身
	- 有两种办法:
		- 第一种是头插法, 就是创建一个新的链表, 然后把原来链表中的值一个一个取出来放到新链表的头部, 最后返回这个链表, 那么就反转了
			```
				class Solution {
				    public ListNode reverseList(ListNode head) {
				        ListNode pre = null; //这个就是新的空链表
				        ListNode cur = head;
				        while(cur!=null){
					        //用来保存当前节点的下一个节点, 用于头插之后更新cur
				            ListNode nxt = cur.next;  
				            cur.next = pre;    
				            // 重新更新空链表的头节点pre, 此时的头节点为cur    
				            pre = cur;          
				            cur = nxt;
				        }
				        return pre;
			    }
			}
			```
