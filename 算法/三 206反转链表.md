-  在链表里面, 返回链表的头节点就是返回链表本身
	- 有两种办法:
		- 第一种是头插法, 就是创建一个新的链表, 然后把原来链表中的值一个一个取出来放到新链表的头部, 最后返回这个链表, 那么就反转了
			```
				class Solution {
				    public ListNode reverseList(ListNode head) {
				        ListNode pre = null; //这个就是新的空链表
				        ListNode cur = head;
				        while(cur!=null){
					        //用来保存当前节点的下一个节点, 用于头插之后更新cur
				            ListNode nxt = cur.next;  
				            cur.next = pre;    
				            // 重新更新空链表的头节点pre, 此时的头节点为cur    
				            pre = cur;          
				            cur = nxt;
				        }
				        return pre;
			    }
			}
			```
		- 第二种是尾插法, 使用递归完成的, 先一步步递到原链表的最后的一个元素, 然后归的时候从最后一个元素慢慢回来改变它们的指针指向的方向
			```
				class Solution {
				    public ListNode reverseList(ListNode head) {
					//首先判断当前链表是否为null或者是否只有一个值, 顺序不能变, 必须按这个顺序来判断, 如果是, 那么就直接返回它本身
			        if(head == null || head.next == null){
			            return head;
			        }
			        //开启递归, 此时的final_node就是当前链表的最后一个节点, final_node不变, 这个就可以看作是一个新的链表了
			        ListNode final_node = reverseList(head.next);
			        //那么就需要一个新的临时变量cur来存储当前的链表的head的下一个值, 因为是根据head.next赋值给final_node的, 那么说明此时的head就是当前链表的最后一个的前一个节点
			        ListNode cur = head.next;
			        //重新构建反向的链表
			        cur.next = head;
			        head.next = null;  //防止最后一次归完后, 头尾节点形成环
			        return final_node;
			    }
			}
			```
