- 第一种, 直接排序, `Arrays.sort(nums)`; 然后返回`nums[nums.length-k]` 就是第k大元素, 但是这个时间复杂度很高

- 第二种, 所以就想到了一个数据结构, 堆, 可以创建一个小顶堆来完成这个问题
	- 思路:先往堆中放入k个元素, 然后从k+1个元素开始与堆顶元素比较, 如果比它大, 就放入堆中, 否则不管
	- 代码实现: 
		```
		class Solution {
		    public int findKthLargest(int[] nums, int k) {
		        // 创建一个小顶堆
		        PriorityQueue<Integer> priorityQueue = new PriorityQueue<Integer>();    
		        for(int i=0; i<k; i++){
		            //将该数组里面的前key个元素添加到小顶堆里面
		            priorityQueue.offer(nums[i]);
		        }
		        // 遍历剩余的k+1个元素
		        for(int i=k; i<nums.length; i++){
		            if(nums[i]>priorityQueue.peek()){
		                priorityQueue.poll();
		                priorityQueue.offer(nums[i]);
		            }
		        }
		        return priorityQueue.peek();
		    }
		}		
		```
- 第三种, 使用快排, 但是需要随机选择基准元素, 通过随机选择可以最大程度上避免了遇到快排最坏的情况, 选到最小或者最大的元素, 导致时间复杂度到了O(n²)
	```
	class Solution {
	    private static final Random rand = new Random();
	    public int findKthLargest(int[] nums, int k) {
	        int n = nums.length;
	         // 第k大元素在升序数组里面的小标就是n-k, 之后只需要判断自己选择的基准元素pivot与targetIndex的关系即可
	        int targetIndex = n - k; 
	        int left = 0, right = n - 1;
	        while(true){
	        
	            // 开始判断基准元素pivot(也就是这里的i)与targetIndex的关系
	            int i = partition(nums, left, right);
	            if(i==targetIndex){
	                //它就是第k大元素
	                return nums[i];
	            }
	            if(i>targetIndex){
	                right = i - 1;
	            }else{
	                left = i + 1;
	            }
	        }
	    }
		
		// 这个函数就是为了返回所选择的基准元素在经过快排后的下标位置
	    private int partition(int[] nums, int left, int right){
	    
	        // 首先随机选择一个基准元素pivot
	        int i = left + rand.nextInt(right-left+1);
	        int pivot = nums[i];
	        
	        // 接下来把基准元素与子数组第一个元素交换, 防止它干扰后续的划分, 因为最后是直接判断排序完后基准元素的下标和上面targetIndex的关系的
	        swap(nums, i, left);
	        
	        // 双向遍历双指针, 将i赋值为left+1,因为在交换之后, i已经没有意义了
	        i = left + 1;
	        
	        int j = right;
	        while(true){
	            while(i <= j && nums[i] < pivot){
	                i++;
	            }
	            // 上面这个循环出来后 nums[i] >= pivot
	            
	             while (i <= j && nums[j] > pivot) {
	                j--;
	            }
	            // 此时 nums[j] <= pivot
	
	    // 还没有相遇的话, 就不会执行这个代码, 如果执行了这个, 就说明各自分区内的排序已经好了
	            if(i>=j){
	                break;
	            }
	            
	            swap(nums, i, j);
	            i++;
	            j--;
	        }
	        
	        // 循环结束了
	        // [ pivot | <=pivot | >=pivot ]
	        //   ^             ^   ^     ^
	        //   left          j   i     right
	        // 状态是上图这样的
	
	        // 接下来将pivot与nums[j]进行交换
	        // 为什么是这样交换的?
	        //
	        swap(nums, left, j);
	        return j;
	    }
	
	    // 交换函数
	     private void swap(int[] nums, int i, int j) {
	        int tmp = nums[i];
	        nums[i] = nums[j];
	        nums[j] = tmp;
	    }
	}
	```
