	- 然后快排需要什么呢, 它需要一个基准元素, 需要左右两个子数组不断递归, 所以就有了方法`partition`去划分子数组.     
	- 首先在`quickSort`方法里面判断当前数组是不是已经有序了, 如果已经有序了, 那么直接返回即可, 否则就进入`partition`方法去划分子数组, 而为了防止时间复杂度变成了`O(n**2)`, 基准元素需要随机选择
		- ```
		  // 首先随机选择一个基准元素
        int i = left + rand.nextInt(right - left + 1);
        int pivot = nums[i];
		  ```
	  选择完基准元素, 将它与当前子数组首元素交换位置, 防止它干扰数组排序, 然后就进入一个循环进行划分, 不过**在标准的双向扫描快速排序中，通常让 `i` 指向第一个大于等于 pivot 的元素，`j` 指向第一个小于 pivot 的元素，交换它们。最后确保 `j` 停在小于 pivot 的位置上，再交换 `left` 和 `j`。** 最后返回基准元素的下标`j`.

```
class Solution {
    private static final Random rand = new Random();

    public int[] sortArray(int[] nums) {
        quickSort(0, nums.length-1, nums);
        return nums;
    }

    private void quickSort(int left, int right, int[] nums){
        // 先判断数组本身是不是升序的
        // 创建一个信号来进行判断
        boolean order = true;

        // 只要有一个左边的元素大于右边的元素, 那么就是没有拍好序, 就使order=false, 然后退出这个循环
        for(int i = left; i < right; i++){
            if(nums[i] > nums[i+1]){
                order = false;
                break;
            }
        }
        if(order){
            return ;
        }

        // 否则就开始划分子数组
        int i = partition(nums, left, right);

        // 左右数组递归进行这个判断
        quickSort(left, i-1, nums);
        quickSort(i+1, right, nums);
    }

    // 定义划分子数组的方法
    private int partition(int[] nums, int left, int right){
        // 首先随机选择一个基准元素
        int i = left + rand.nextInt(right - left + 1);
        int pivot = nums[i];

        // 将基准元素与子数组第一个元素交换位置, 防止它影响划分
        swap(nums, i, left);

        // 双向指针(i, j)遍历子数组使得基准元素两边各自有序
        i = left + 1;
        int j = right;
        while(true){
            while(i <= j && nums[i] < pivot) i++; // 此时nums[i] > pivot
            while(i <= j && nums[j] > pivot) j--; // 此时nums[j] < pivot
            if(i >= j){
                break;
            }

            // 如果没有退出循环, 那么就交换这两个值
            swap(nums, i, j);
            i++;
            j--;
        } 
        
        // 上面循环结束后, 刚好j在i的左边, 那么将pivot与nums[j交换位置, 此时pivot的小标为left, 从第一次交换完就没有变化
        swap(nums, left, j);
        return j;
    }

    public void swap(int[] nums, int i, int j){
        int ans = nums[i];
        nums[i] = nums[j];
        nums[j] = ans;
    }
}
