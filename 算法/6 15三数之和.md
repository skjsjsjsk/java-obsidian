想法就是, 先把数组排序了, 固定`nums[i]`, 如果使用双指针去遍历剩余的元素看是否等于0

[可以用Arrays.asList(A, B, C)或者List.of(A, B, C), 把这些元素变成一个数组添加到另一个数组中, 不够由List.of创建出来的数组不可变]
```
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        // 排序
        Arrays.sort(nums);
        //创建一个新数组用来返回结果
        List<List<Integer>> result = new ArrayList<>();
        
        for (int i = 0; i < nums.length - 2; i++) {
            // 先去重, 判断此刻的nums[i]是否与前一个相同, 如果相同就跳过, 因为已经判断过相同情况了
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            
            // 两个优化
            // 三个最小的加起来依旧大于0, 那么肯定不会有等于0的情况
            if(nums[i] + nums[i+1] + nums[i+2]>0) break; 
            // 与最大的两个相加小于0的话, 说明还有可能会等于0, 因为nums[i]还可以继续变大
            if(nums[i]+nums[nums.length-1]+nums[nums.length-2]<0) continue;

            // 如果不相同, 那么就设置一个双指针, 用来遍历除nums[i]以外的其他元素
            // 左指针为i+1, 这样就保证是从nums[i]之后的那个元素开始的
            int left = i + 1, right = nums.length - 1;

            // 开启循环判断
            while (left < right) {
                int threeSum = nums[i] + nums[left] + nums[right];

                // 先判断threeSum是否等于0, 等于则把这组结果添加到数组中
                if (threeSum == 0) {
                    result.add(List.of(nums[i], nums[left], nums[right]));

                    // 接着判断左右指针是否有重复情况再进行下一轮循环前
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    
                    // 上面的重复判断完成之后, 当前的left和right的位置都位于最后一个重复元素的位置, 所以需要让左指针再向左移动一位, 右指针向右
                    left++;
                    right--;
                } else if (threeSum < 0) {
                    left++;
                } else {
                    right--;
                }
            }
        }
        return result;
    }
}
```
